Le régime unique de la haie exclut de son champ d'application ( art. L412-21 du code de l'environnement) :

  • les haies en bordure de bâtiment
  • les haies en bordure de place publique
  • les haies dans un parc ou jardin attenant à une habitation, ou en formant l'enceinte
  • les alignements d'arbres

Par souci de cohérence pour l'usager, et pour éviter les sous-déclarations, le simulateur couvre la réglementation pour tous les linéaires, y compris les alignements d'arbres.

Ainsi, trois catégories juridiques de linéaires se présentent :

  • les haies régime unique {% include 'haie/petitions/_hedge_category.html' with category=HedgeCategory.ru %}
  • les alignements d'arbres L350-3 {% include 'haie/petitions/_hedge_category.html' with category=HedgeCategory.l350_3 %}
  • les autres linéaires : haies hors régime unique et alignements d'arbres non L350-3 {% include 'haie/petitions/_hedge_category.html' with category=HedgeCategory.hru %}

Le simulateur invite le demandeur à déposer un dossier distinct pour chaque catégorie (Prochainement, le dépôt de plusieurs dossiers se fera simultanément). Pour les linéaires autres que les alignements d'arbres L350-3, le dépôt se fait systématiquement sur ce portail. Pour les alignements d'arbres L350-3, le canal de dépôt est au choix de la DDT(M) (voir page de paramétrage).

Pour déterminer ces catégories, le simulateur s'appuie sur les déclarations du demandeur.

L'instructeur du service coordonnateur est donc invité à vérifier :

  • si le type « alignement d'arbres » est correctement déclaré. Il convient notamment de veiller à la possible ambiguïté avec le type « haie arborée », plus adapté si une deuxième strate végétale est présente ;
  • si les caractéristiques excluant une haie du régime unique sont correctement déclarées : la haie est-elle bien en bordure de bâtiment ou de place publique, est-elle bien située dans un parc ou jardin attenant à une habitation ?
