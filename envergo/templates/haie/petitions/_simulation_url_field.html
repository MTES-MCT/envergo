{% comment %} Field wrapper for the alternative simulation url. Same layout as the default field template, but when the url is an invalid simulation it lists, below the field, the headline error and every underlying moulinette error (exposed as `moulinette_errors`). The stock template only shows the first error, hence this dedicated wrapper. Several messages use DSFR's messages group (`.fr-message--error` paragraphs), not a single `.fr-error-text`. {% endcomment %}
{% include 'django/forms/fields/label.html' %} {{ field }} {% if field.errors %}
{% for error in field.errors %}

{{ error }}

{% endfor %} {% for error in moulinette_errors %}

{{ error }}

{% endfor %}
{% endif %}
